# Aplicación simple de notas hecha con Express js y Handlebars.

## Variables de entorno
```
PORT= Puerto de escucha de express
MONGODB_URI= URI para conexión con MongoDB
SECRET= Clave secreta para cifrar las sesiones
```

## Uso
```bash
$ npm run install
```
```bash
$ npm start
```

## Uso en desarrollo
```bash
$ npm run dev
```
